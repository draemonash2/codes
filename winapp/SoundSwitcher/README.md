# Sound Switcher

Windows 11 のサウンド設定の「再生」と「録音」を、ひとつのコンパクトなウィンドウから
まとめて確認・切り替えできる常駐ユーティリティです。

## 概要

- **再生／録音デバイスを一画面で表示** — Windows の「サウンド」設定を開かずに、
  既定デバイスの確認と切り替えがワンクリックでできます。
- **未接続のペアリング済み機器も表示** — 接続中の機器に加え、ペアリング済みだが
  未接続の機器もグレー表示で一覧されます。
- **機器種別アイコン** — スピーカー／ヘッドホン／ヘッドセット／ディスプレイ(HDMI/DP)／
  マイクを、Windows のサウンドパネルのようにアイコンで出し分けます。
- **バッテリー残量表示** — バッテリー駆動の Bluetooth 機器は、アイコン周囲の色付き円弧
  で残量を表示します（残量を OS に報告する機器のみ）。表示中は約30秒ごとに自動更新します。
- **音量スライダー（出力／入力）** — ウィンドウ左端に出力・入力それぞれの縦型音量バーを
  備え、上下ドラッグで音量を調整できます。一番下までドラッグするとミュートになります。
- **レベルメーター** — 各音量スライダーの右横に信号レベルメーターを表示できます（右クリック
  メニューで表示切替、設定は記憶。**既定はオフ**）。出力は再生中の音に常時反応します。入力
  （マイク）は、メーター表示のためにマイクを占有しない方針のため、他アプリがマイクを使用して
  いる間のみ反応します。
- **デスクトップ常駐** — 通知領域（システムトレイ）に常駐。タスクバーや Alt+Tab
  （タスク切り替え）には表示しません。
- **軽量・高速** — メモリ使用量を抑え（常駐時 約20MB）、起動も高速です。

## 使い方

1. `SoundSwitcher.exe` を実行します。
   - 初回に **.NET 8 デスクトップ ランタイム** が無い場合は、インストールを促す
     ダイアログが表示されます。「はい」を選ぶと自動でインストールします
     （winget 使用。無い場合は公式ダウンロードページを開きます）。
2. ウィンドウ上部が**再生(PLAYBACK)**、下部が**録音(RECORDING)** の機器一覧です。
3. **機器カードをクリック**すると、その機器が既定デバイスに切り替わります
   （既定の機器は青い枠で表示）。
4. **ウィンドウ左端の音量バー** — 上が出力、下が入力です。バーを上下にドラッグして
   既定デバイスの音量を調整します。一番下まで下げるとミュートになります。各バーの右横は
   現在の信号レベルメーターです（出力は再生中の音、入力は他アプリのマイク使用中に反応）。
5. **ウィンドウの移動** — 余白部分をドラッグします。
6. **ウィンドウのサイズ変更** — 端をドラッグします。位置とサイズは次回起動時に復元されます。
7. **ウィンドウを右クリック** — メニューから以下を操作できます。
   - **サウンド設定** — Windows の「サウンド」設定を開きます。
   - **Bluetooth設定** — Windows の「Bluetooth とデバイス」設定を開きます。
   - **レベルメーターを表示** — レベルメーターの表示／非表示を切り替えます（設定は記憶。
     既定はオフ。非表示時はメーター分のスペースが詰まります）。
   - **終了** — アプリを終了します。
8. **トレイアイコン** — 左クリックで表示／非表示、右クリックで「表示/非表示・終了」。

## 動作環境

- Windows 10 / 11 (x64)
- .NET 8 デスクトップ ランタイム（未導入時は起動時にインストールを案内）

## 構成

```
SoundSwitcher.exe          起動用ランチャー（ランタイム確認・導入を行い本体を起動）
app\SoundSwitcherApp.exe   本体アプリ
```

## 変更履歴

### 2026-06-05
- レベルメーターを既定オフに変更し、右クリックメニューで表示切替できるように（設定を記憶）。
  非表示時は CPU 負荷ゼロ、かつメーター分のスペースを詰めてカードを左寄せ表示。
- レベルメーターの描画を軽量化し（単色フィル＋縦スケール、更新頻度を抑制）、配色を
  スライダーと同系色（落ち着いたグレー）に統一。
- バッテリー残量を、表示中は約30秒ごとに自動更新（手動「更新」メニューは廃止）。
- バッテリー残量の数値（%）表示を廃止し、円弧リングのみ表示に。
- 右下の歯車ボタンを廃止し、右クリックメニューに「サウンド設定」を追加。
- 右クリックメニューに「Bluetooth設定」を追加。
- 通知領域（トレイ）アイコンの色を少し明るく調整。

### 2026-06-04
- ウィンドウ左端に出力／入力の縦型音量スライダーを追加（上下ドラッグで音量調整、
  最下部までドラッグでミュート）。
- 各音量スライダーの右横にレベルメーターを追加（出力は再生音に常時反応、入力は
  他アプリがマイクを使用している間のみ反応。メーターのためのマイク占有はしない方針）。
- ウィンドウを Alt+Tab／タスクビューに表示しないよう変更（ツールウィンドウ化）。

### 2026-06-03
- 配布形態をフレームワーク依存（FD）に変更し、サイズを約 179MB → 約 12MB に削減、
  初回起動を大幅に高速化。
- ランチャーと本体のファイル名を分離（本体を `app\SoundSwitcherApp.exe` に）。
- ランタイム未導入時にインストールを案内・自動実行するランチャーを追加。
- 右下に「Windows サウンド設定を開く」ボタンを追加。
- 既定機器アイコンの青丸を廃止（青枠のみで表示）。
- 終了時に通知領域アイコンが残らないよう改善。
- 機器種別（スピーカー／ヘッドホン／ディスプレイ／マイク等）でアイコンを出し分け。
- フォントを Meiryo UI に変更。タスクマネージャー等に表示されるアプリアイコンを設定。
- Bluetooth 機器のバッテリー残量取得に対応（SetupAPI / DEVPKEY_Bluetooth_Battery）。
- 通知領域（システムトレイ）常駐に対応し、タスクバー非表示に変更。
- UI を簡素化（タイトルバー・セクション見出し・非接続バッジを廃止、全体ドラッグ移動）。
- メモリ使用量を削減（ソフトウェアレンダリング化、常駐時のワーキングセット圧縮）。
- ウィンドウのリサイズ対応と、位置・サイズの記憶を追加。

### 初版
